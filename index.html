API Downloader (DEV)

request 
  post : http://cabin.iooo.pw:3000/file/find
  body : {"uid":"1", "akey":"2"}

response : 
{
  "fileUrl":"http://path/to/fileName.dat",
  "success":true,
  "msg":"안내 메시지"
}

사용법

  1. API Data를 신청
  2. 승인후 다운로드 인증키를 발급(500MB 이상)
  3. 신청아이디와 인증키를 좌측 폼에 입력
  4. Download 버튼을 클릭한다.
  5. 입력값 유효성 체크 후 다운로드 시작.

참고

  • 다운로드 중 연결이 끊기거나 종료된경우 재시도 하면 이어받기가능
